# Cz-Portal-Front-Shop

## Introduction

This project is the front page for Cz-Portal's shop

##stack
* reactjs
* react-router4
* redux
* react-redux
* react-router-redux
* redux-saga
* semantic-ui

## Install

    $ git clone https://github.com/jomalonejia/Cz-Portal-Front-Shop.git
    $ cd Cz-Portal-Front-Shop
    $ npm install


## Start

    $ npm start

## Build

    $ npm run build
